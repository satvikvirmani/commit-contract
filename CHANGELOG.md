# Changelog

All notable changes to Commit Contract are documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [1.0.0] - 2026-03-31

### Added
- Setup command to configure Copilot commit message instructions for Conventional Commits.
- Status bar UX with clear setup state and quick action access.
- Optional auto-setup behavior (`commitStyle.autoSetupOnOpen`) once per workspace.
- Custom style instruction support via `commitStyle.customInstructionText`.
- Workspace output channel logging (`Commit Contract`) for setup and diagnostics.
- Publish metadata and marketplace assets (icon, license, documentation).
